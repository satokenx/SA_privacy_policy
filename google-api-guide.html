🔑 Google API Key Guide

Google API キー取得ガイド

1

Access Google AI Studio

Access Google AI Studio Google AI Studio にアクセス

Open Google AI Studio →

Please log in with your Google account. Googleアカウントでログインしてください。

2

Create API Key

Click "Create API Key" button to generate a new API key. 「Create API Key」ボタンをクリックして、新しいAPIキーを生成します。

3

Copy API Key

Copy the generated API key and save it in a secure location. 生成されたAPIキーをコピーして、安全な場所に保存してください。

⚠️ Important Notes

• API keys are confidential. Do not share with others
• Delete unused API keys
• Charges may apply based on usage • APIキーは秘密情報です。他人と共有しないでください
• 不要になったAPIキーは削除してください
• 使用量に応じて料金が発生する場合があります

📝 How to Use

The obtained API key can be used with various Google AI services (Gemini API, PaLM API, etc.). 取得したAPIキーは、Google AI の各種サービス（Gemini API、PaLM API など）で使用できます。

Save the API key in environment variables or configuration files for use in your applications. APIキーを環境変数や設定ファイルに保存して、アプリケーションから利用してください。
